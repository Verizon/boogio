'''
cleanup.py

Clean up selected files in a directory based on filename timestamp and
selected cleanup period.

All files older than the start of the previous cleanup period will be
deleted, unless their timestamp is the earliest in their cleanup
period.

For example, if the cleanup period is 'hour', and the current time is
0420, all files whose timestamp is earlier than 0300 will be deleted
except for the files whose timestamps are closest to the start of the
hour in which they fall.

If the timestamps are at 0216, 0230, 0245, 0255, 0315, 0330, 0340,
0402 and 0414, an 0420 'hour' cleanup will remove the files with
timestamps 0230, 0245 and 0255, leaving 0216, 0315, 0330, 0340, 0402
and 0414. Any additional cleanups before 0500 won't remove any more
files. A subsequent 0500 cleanup will remove 0330 and 0340, leaving
0216, 0315, 0402 and 0414.

Summarizing, if the file timestamps in each hour are as follows:

    02xx:   0216, 0230, 0245, 0255
    03xx:   0315, 0330, 0340
    04xx:   0402, 0414

an 'hour' cleanup in the 04xx hour window will remove all but the
earliest file in the 02xx window and any earlier window.


'''

        '''
        )

    parser.add_argument(
        '-v', '--verbose',
        dest='verbose',
        default=False,
        action='store_true',
        help='''show more output. '''
        )

    parser.add_argument(
        '-p', '--period',
        choices=['hour', 'day', 'week'],
        default='hour',
        help='''time period for cleanup. '''
        )

    return parser
